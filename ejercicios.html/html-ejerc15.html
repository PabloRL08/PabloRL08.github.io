Tipos de selectores que entran en examen

Clase (class="")

Este es un ejemplo de un párrafo con una clase aplicada.

El selector de clase se utiliza para seleccionar elementos HTML que tienen un atributo de clase específico. Es muy útil para aplicar estilos a múltiples elementos sin necesidad de asignar un identificador único a cada uno. En el ejemplo anterior, todos los elementos con la clase "mi-clase" tendrán el texto de color azul.

Identificador (id="")

Este es un ejemplo de un párrafo con un identificador único.

El selector de identificador se utiliza para seleccionar un elemento único que tiene un atributo id específico. Esto es útil cuando se desea aplicar estilos a un solo elemento en la página. En el ejemplo anterior, el párrafo con id "mi-id" tendrá el texto en negrita.

El selector universal (*) selecciona todos los elementos en el documento HTML. Es útil para aplicar un estilo global a todos los elementos, aunque se debe usar con precaución para no afectar el rendimiento o la especificidad de otros estilos.

Sobrevuelo (p:hover)

Coloca el cursor sobre este párrafo para ver el efecto de hover.

El selector de hover se usa para aplicar estilos a un elemento cuando el mouse pasa sobre él. En el ejemplo, cuando el cursor está sobre el párrafo, el texto cambiará a color rojo. Esto es útil para mejorar la interactividad de la página y resaltar elementos importantes.
